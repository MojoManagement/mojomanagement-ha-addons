# MojoManagement Home Assistant Add-ons

This folder contains the structure for a Home Assistant custom add-on repository.

## Included add-ons

- `yt-cast-audio-bridge`

## Publishing in a dedicated repository

If you host add-ons in a dedicated repository such as `mojomanagement-ha-addons`, copy this folder content to that repository root:

- `repository.yaml`
- `yt-cast-audio-bridge/`

Then add that repository URL in Home Assistant Add-on Store.
