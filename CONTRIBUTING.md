# Contributing

Thanks for contributing.

## Development

```bash
npm install
cp .env.example .env
npm run lint
npm test
npm run smoke
```

## Scope

- Root `package.json` commands proxy into `yt-cast-audio-bridge/app`.
- Home Assistant add-on packaging files are in `yt-cast-audio-bridge/`.

## Commit style

Use clear, imperative commit messages.
